🔥 Step 1 of 4
Create Firebase Account
Firebase is Google's free database. All your data will be securely stored here.
📌 Steps:
1. console.firebase.google.com — Go to this link
2. Sign in with Gmail
3. Click "Add project"
4. Enter project name: sunstar-solutions
5. Disable Google Analytics
6. Click "Create project"
🗄️ Step 2 of 4
Enable Database & Auth
Two Firebase services need to be enabled.
Enable Authentication:
1. Left menu Authentication → Get started
2. Email/Password → Enable → Save

Enable Realtime Database:
3. Left menu Realtime Database → Create database
4. Location: us-central1 (default)
5. Test mode — keep this for now → Enable
⚠️ Database Rules — Go to Rules tab and paste the code below:
{ "rules": { "users": { "$uid": { ".read": "auth != null && ($uid === auth.uid || root.child('users').child(auth.uid).child('role').val() === 'super_admin')", ".write": "auth != null && ($uid === auth.uid || root.child('users').child(auth.uid).child('role').val() === 'super_admin')" } }, "companies": { ".read": "auth != null", ".write": "auth != null && root.child('users').child(auth.uid).child('role').val() === 'super_admin'" }, "visits": { "$cid": { ".read": "auth != null && (root.child('users').child(auth.uid).child('company_id').val() === $cid || root.child('users').child(auth.uid).child('role').val() === 'super_admin' || root.child('users').child(auth.uid).child('role').val() === 'company_admin')", ".write": "auth != null && root.child('users').child(auth.uid).child('company_id').val() === $cid" } } } }

4. Click Register app
5. Copy the firebaseConfig object below
